Quizify Pro (A dynamic quiz web app)

Quizyfy Pro is a web application designed for educational institutes to create and manage online MCQ exams efficiently. It provides tools for instructors to design exams, customize question banks, and automate grading, streamlining the assessment process. With secure login, real-time scoring, and detailed analytics, Quizyfy Pro enables educators to administer tests seamlessly and evaluate student performance with accuracy, making it ideal for modern online education.

Speech to ISL converter web app

Speech to ISL Converter is a web application designed to enhance accessibility by translating spoken or typed words into Indian Sign Language (ISL). Aimed at bridging communication gaps for the Deaf and Hard of Hearing community, the app converts input into animated ISL gestures, allowing for real-time communication and learning.
